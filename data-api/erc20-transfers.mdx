This guide will walk you through the process of listing all ERC-20 transfers associated with a specific wallet address on the DFK L1 network using the Data API.

### Step 1: Set Up AvaCloud

First, ensure that you have set up [AvaCloud](https://app.avacloud.io/) and have access to your API key. If youâ€™re new to Avacloud, create an account and obtain an API key.

### Step 2: Retrieve the Native Balance of an Address

To obtain a list of ERC-20 transfers you can use the [List ERC-20 transfers](/data-api/evm-transactions/list-erc-20-transfers) endpoint. Youâ€™ll need to specify the `chainId` and the `address` for which you want to retrieve the balance.

Hereâ€™s how you can do it:


<CodeGroup>
```javascript AvacloudSDK
import { AvaCloudSDK } from "@avalabs/avacloud-sdk";

const avaCloudSDK = new AvaCloudSDK({
  apiKey: "<YOUR_API_KEY_HERE>",
  chainId: "53935",
  network: "mainnet",
});

async function run() {
  const result = await avaCloudSDK.data.evm.transactions.listErc20Transactions({
    pageSize: 10,
    address: "0x1137643FE14b032966a59Acd68EBf3c1271Df316",
  });

  // Handle the result
  console.log(JSON.stringify(result, null, 2));
}

run();
```
